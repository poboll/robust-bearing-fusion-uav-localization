# Journal Scope Notes

## Engineering Applications of Artificial Intelligence

Source:
- https://www.sciencedirect.com/journal/engineering-applications-of-artificial-intelligence

Relevant scope signals:
- emphasizes practical AI methods for real-world engineering applications
- asks for validated and replicable work
- explicitly warns that new metaphor-based metaheuristics are rarely accepted

Implication for this project:
- strong fit only if the paper is written as a robust engineering localization framework
- weak fit if written as “a new heuristic optimizer”

## Applied Soft Computing

Source:
- https://www.sciencedirect.com/journal/applied-soft-computing

Relevant scope signals:
- focuses on soft computing for real-life problems
- includes robotics, sensor systems, signal processing, and hybrid methods
- welcomes robust, uncertainty-tolerant, practical methods

Implication for this project:
- good fit if the paper highlights robust estimation, hybrid geometric-refinement design, and degraded sensing
- moderate risk if the manuscript underplays the soft-computing angle entirely

## Expert Systems with Applications

Source:
- https://www.sciencedirect.com/journal/expert-systems-with-applications

Relevant scope signals:
- focuses on expert and intelligent systems design, development, testing, and application
- discourages metaphor-based algorithm repackaging
- emphasizes genuine innovation and clear explanation

Implication for this project:
- potentially strong fit if the paper emphasizes interpretable intelligent estimation under realistic degraded sensing
- should avoid sounding like a pure localization geometry note with weak systems framing

## Best Practical Interpretation

The current project is most aligned with journal scope when framed as:
- an interpretable robust intelligent localization framework
- validated under realistic degraded measurement conditions
- competitive with heuristic baselines but not built around a heuristic gimmick
