# Submission Checklist V1

## Manuscript

- [ ] finalize target journal
- [ ] select journal-specific title
- [ ] select journal-specific abstract
- [ ] polish manuscript_v3 into final submission text
- [ ] standardize citation format
- [ ] remove all internal drafting notes

## Figures and Tables

- [ ] insert Figure 1-4 into final manuscript
- [ ] insert Tables 1-6 into final manuscript
- [ ] check caption wording for journal style
- [ ] verify units, rounding, and significant digits
- [ ] ensure figure fonts are journal-friendly

## Scientific Scope Check

- [ ] no universal claim over PSO
- [ ] no overclaim of real-world deployment
- [ ] no overstating bias estimation as the sole innovation
- [ ] clearly state simulation-based limitation
- [ ] clearly position scheduling/recovery as future work or secondary extension

## Journal Packaging

- [ ] choose one cover letter variant
- [ ] adapt manuscript tone to journal scope
- [ ] prepare highlights if required
- [ ] prepare graphical abstract if required
- [ ] check data/code availability statement requirement

## Final Technical Check

- [ ] rerun core experiments before submission freeze
- [ ] archive JSON outputs used in tables
- [ ] verify figure files match manuscript numbers
- [ ] confirm runtime and ablation values after final code freeze
