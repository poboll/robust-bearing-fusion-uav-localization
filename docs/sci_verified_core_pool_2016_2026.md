# SCI/SCIE Verified Core Pool (2016-2026)

## 1. Scope and Verdict

- Raw literature pool: `79` real records with title, venue, year, DOI or landing page traceability.
- Strict `SCI/SCIE` core pool: `37` journal papers.
- Core pool year span: `2019-2026`.
- Supplement pool: `15` recent and useful frontier papers, but not all suitable as `SCI` backbone evidence.
- Downgraded / excluded pool: `27` papers kept only as background inspiration or local problem-context references.

## 2. Verification Rule

The core pool uses a conservative venue-level rule:

1. The paper must be a real journal paper with a resolvable DOI or stable landing page.
2. The paper must fall in the recent-decade window required for this project.
3. The venue must belong to a conservative whitelist of journal outlets that are appropriate for `SCI/SCIE`-level argumentation in this topic.
4. Ambiguous venues are downgraded instead of being force-labeled as `SCI`.

### Journal-indexing notes re-checked in this round

- `Sensors`: official MDPI journal history page states it is covered by `SCIE`.
- `Drones`: official MDPI journal history page states it was accepted for inclusion in `SCIE`.
- `Biomimetics`: official MDPI journal history page states it is covered by `SCIE`.
- `Computers`: official MDPI history page shows inclusion in `ESCI`, so it is **not** used as strict `SCI/SCIE` core evidence.

### Practical implication

- The old `79`-paper pool is useful as a broad search pool.
- The actual paper-writing backbone should switch to the `37`-paper core pool below.
- Any paper from `Computers`, arXiv, TechRxiv, repository pages, or conference proceedings should be treated as `frontier supplement`, not as the central `SCI` evidence chain.
